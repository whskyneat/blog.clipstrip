== Installation ==

- Install from WP Dashboard>>Plugins>>Add new plugin>>Search for `SocialRadios' plugin. Activate the plug-in. WP Dashboard>>Appearance>>Widgets: Drag&drop SociaRadios to 'SocialRadios Sticky' section(header or footer mode) or to your column section.  
WP Dashboard>>SocialRadios: Activate ajax-refresh, a new function that allows your users to listen and navigate at the same time. 
[youtube https://www.youtube.com/watch?v=WbMctGPd-JQ] 

- Instala desde WP Dashboard>>Plugins>>Añadir nuevo>>Busca el plugin `SocialRadios'. Actívalo. Configuralo en WP Dashboard>>Apariencia>>Widgets.
Arrastra SocialRadios a 'SocialRadios Sticky' sección (Modo cabecera o pie de página) o a una columna.  
WP Dashboard>>SocialRadios: Activa ajax-refresh, una nueva opción para que los visitantes puedan escuchar la radio al mismo tiempo que visitan tu blog.
